Zahlen-Schieber

Schiebe, verschmilz, verdopple: Aus 2 und 2 wird 4, aus 1024 irgendwann die große Zielkachel. Halb Pause, halb Kopfrechnen.

So funktioniert's

Jeder Zug schiebt alle Kacheln bis zum Anschlag in eine Richtung. Treffen dabei zwei gleiche Zahlen aufeinander, verschmelzen sie zu ihrer Summe — jede Kachel aber höchstens einmal pro Zug. Aus 2 + 2 wird also 4, aus 64 + 64 wird 128.

  • Punkte: Jede Verschmelzung bringt so viele Punkte, wie die neue Kachel wert ist.
  • Nachschub: Nach jedem gültigen Zug erscheint eine neue Kachel auf einem freien Feld — meist eine 2 (90 %), manchmal eine 4 (10 %).
  • Ende: Ist das Feld voll und keine zwei Nachbarn sind gleich, geht nichts mehr. Die 2048 ist das klassische Ziel — danach läuft das Spiel einfach weiter.
  • Taktik: Parke deine größte Kachel in einer Ecke und vermeide eine der vier Richtungen so gut es geht — dann bleibt das Brett aufgeräumt.

Nebenbei ein bisschen Mathematik: Alle Werte sind Zweierpotenzen — die Zielkachel ist 211 = 2048. Die Bestenliste (Top 10) wird nur in deinem Browser gespeichert.
