Bingo Beamermodus

Eine Ziehungsmaschine für die digitale Tafel: Wählt einen Pool, dann wird mit jedem Knopfdruck zufällig ein Wert ohne Wiederholung gezogen. Die zuletzt gezogene Zahl bzw. der Begriff erscheint riesig, darunter wächst ein Verlaufsraster aller Ziehungen. Alle haben eigene oder gedruckte Bingo-Karten und markieren mit. Tipp: oben rechts auf Vollbild schalten.

So funktioniert's

Jede:r braucht eine Bingo-Karte — selbst gemalt oder gedruckt. Ein 5×5-Raster, die Mitte ist frei. Wählt an der Tafel den Pool (Zahlen 1–75 oder allgemeine Begriffe) und tippt auf „Nächste Ziehung". Der gezogene Wert wird groß angezeigt und ins Verlaufsraster aufgenommen; wer ihn auf seiner Karte hat, kreuzt ihn an. Wer zuerst eine volle Reihe, Spalte oder Diagonale hat, ruft „Bingo!". Mit „Neues Spiel" wird der Pool neu gemischt, und mit „Zufalls-Spielkarte" könnt ihr euch zur Übung eine Beispielkarte erzeugen lassen.

Alles läuft auf einem Gerät (der Tafel) — es wird nichts gespeichert und nichts ins Netz geschickt.
