Symmetrie-Werkstatt

Zwei Arten von Symmetrie zum Ausprobieren. Im Modus 0 wird eine asymmetrische „L“-Figur an einer senkrechten Spiegelachse x = s gespiegelt: Verschiebe die Achse mit dem Regler und beobachte, wie das Spiegelbild mitwandert. Jeder Punkt (x | y) wird dabei auf (2s − x | y) abgebildet — die Höhe bleibt, der Abstand zur Achse klappt um. Im Modus 1 erscheint ein regelmäßiges Vieleck mit allen seinen Symmetrieachsen: Wähle die Eckenzahl n und lies ab, dass ein regelmäßiges n-Eck genau n Symmetrieachsen besitzt und sich beim Drehwinkel 360°/n auf sich selbst abbildet (Drehsymmetrie der Ordnung n).
